# Smart Real Estate Management System

智慧售楼管理系统，采用前后端分离架构。

## Architecture

- Backend: Spring Boot, Spring MVC, MyBatis, MySQL, JWT, WebSocket, Knife4j, Maven
- Admin Web: Vue3, Vite, Element Plus, Pinia, Axios, ECharts
- Mobile Apps: uni-app, Vue3, Pinia, uni.request, uni.connectSocket

## Modules

- 管理后台：楼盘管理、楼栋管理、房源管理、销售管理、用户管理、公告管理、预约管理、数据统计、操作日志
- 销售端：房源浏览、预约确认、公告查看、个人中心
- 用户端：注册登录、房源浏览、收藏房源、预约看房、公告查看、个人中心

## Development Principle

按模块逐步开发，每个阶段完成并确认后再进入下一阶段。
